Mode d'emploi

  1. Réponses correctes attendues :
    • Exo 1 : is grown / be planted / is being produced / wants / are being developed
    • Exo 2 : Rice is grown by farmers
    • Exo 3 : les 3 premières (Indonesia produces / year-round / Scientists)
    • Exo 4 : for / ago / since
    • Exo 5 : harvested / will be available
    • Exo 6 (dictée audio) : Excuse me / search / turn around — nécessite Gemini (via fallback automatique si autre provider actif)
  2. Pour utiliser cette page, charge-la via une URL Gymglish (impossible en local). Alternative simple : ajoute temporairement "file:///*" dans manifest.json sous matches et host_permissions, recharge l'extension, puis ouvre cette page en file://.
